==== MATLAB 7.0.x ( R14SP1, R14SP2, R14SP3 ) ====
  *[[http://www.mathworks.com/academia/student_version/doc_r14.html|MathWorks "MATLAB & Simulink Student Version Release 14 with Service Pack 3, Documentation"]].
  *Book "MATLAB and Simulink Student Version (Paperback)" = MATLAB R14, SP1.
    *[[http://www.amazon.de/Matlab-Simulink-Student-Version-Release/dp/0975578782/|Amazon.de Marketplace "Matlab and Simulink Student Version Release 14. CD-ROM"]].
    *[[http://www.amazon.de/MATLAB-Simulink-Student-Version-Release/dp/0975578723|Amazon.de Marketplace "MATLAB and Simulink Student Version: Release 14"]].
    *[[http://www.amazon.com/Matlab-Simulink-Release-Student-Version/dp/0975578731/|Amazon.com Marketplace "MATLAB and Simulink Student Version Release 14"]].
  *MATLAB 7.0.x ( R14SP2 ) and older - "CD-ROM must be in CD-ROM drive to launch MATLAB and view online documentation". The full version for Windows has no such copy protection, while the full version for Linux uses a FLEXlm license manager.
  *MATLAB 7.1, SIMULINK 6.3 (R14SP3) and newer - Mandatory "Software activation" for the student edition. The CD-ROM must not be in the CD-ROM drive to launch MATLAB anymore.
  *MATLAB R14SP1 release (student version) supports Data Acquisition Toolbox 2.5, and R14SP3 release (student version) supports Data Acquisition Toolbox 2.7. Neither DAQ 2.8 nor 2.9 are supported in these two releases.
  *The student edition of Simulink is limited to 1000 blocks !
